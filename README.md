# ☀️ Tenki

Aplicação web para consulta de clima desenvolvida com **React** e **TypeScript**.

---

## 🚀 Como subir a aplicação?

Use:

```sh
docker run -p 80:80 -d hzokbe/tenki:latest
```

> 💡 Notas:
>
> - Por padrão, a aplicação estará disponível em `http://localhost:80/`

---

## 🛠️ Tecnologias

- **TypeScript**
- **React**
- **Docker**
